# AICTE Oasis Infobyte Internship Projects (OIBSIP)

Welcome to my Data Science Internship repository! This repository contains three comprehensive projects completed as part of the **AICTE Oasis Infobyte Internship Program**. This opportunity is designed to provide practical exposure, industry-level experience, and skill development through structured tasks.

Each task demonstrates practical applications of data science, ranging from exploratory data analysis (EDA) to building machine learning classification and regression models.

## 📁 Repository Structure

The repository is organized into three distinct folders, one for each task. Each folder is self-contained with its respective dataset, Python script, and a specific README explaining the methodology.
